Custom Keybinding used in @emacs_gifs

Where possible (i.e. I remember/stop myself at the time) I will use default Emacs keys only.

Of course, old habits die hard and my customisations to Emacs are very extensive, so there's going to be several shortcuts that I use that don't appear in your Emacs defaults.

My main custom keys live in here here

The full config used in @emacs_gifs is ocodo's .emacs.d

I plan to add a nice quick reference chart to show the keys I use. I'm also thinking about using Emacs key capture facilities to make quick reference guides for each GIF I record. Thankfully Emacs makes doing this MUCH easier than normal text editors.

Follow the @emacs_gif twitter feed, I'll make sure to send a message when this page is more helpful!
